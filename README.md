# Shivam Kumar Kesharwani - Full-Stack Developer Portfolio

Welcome to the repository for my personal portfolio website! This project is designed to showcase my skills, experience, and the real-world applications I've built. The design focuses on a modern, dark-themed aesthetic with smooth animations and a responsive layout.

## 🚀 Live Demo

**[View the Live Portfolio](https://shivam-kumar-kesharwani.onrender.com/)**

## ✨ Features

- **Modern UI/UX:** A sleek, dark-themed interface with vibrant green accents.
- **Responsive Layout:** fully optimized for desktops, tablets, and mobile devices (tested across 1440px, 800px, 450px, and 360px breakpoints).
- **Smooth Animations:** Integrated with GSAP for high-performance scroll and reveal animations.
- **Dynamic Projects Section:** Clean presentation of my latest full-stack projects including the SISTec Alumni Portal.
- **Interactive Contact Form:** A visually appealing contact section with interactive elements and a functional message submission system.

## 🛠️ Technologies Used

### Frontend
- **React.js** (via Vite)
- **Vanilla CSS** & **Tailwind CSS** methodologies
- **GSAP** (GreenSock Animation Platform)

### Backend (Contact Form)
- **Node.js / Express.js**
- **REST APIs**

## 📦 Local Development Setup

To run this project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Shivam774705/new_portfolio.git
   cd new_portfolio/port
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`.

## 📂 Project Structure

- `/src/components` - Contains all modular UI components (Hero, About, Projects, Skills, Contact, Footer).
- `/src/assets` - Stores static media like images and icons.
- `/src/index.css` - Global stylesheet including the dark theme variables and comprehensive responsive breakpoints.
- `/src/App.jsx` - Root application component that handles the layout, navigation routing, and GSAP scroll triggers.
